# Agent Network v0.2.2

Private HTTP/JSON client release.

## SDK Changes

- Added `createGalHttpJsonAgentClient(...)` for authenticated Agent Card lookup
  and delegated task creation against deployed product/service agents.
- Added `GalHttpJsonAgentClientError` so callers can preserve status, stable
  error code, safe response body, and `Retry-After` metadata.
- Documented the HTTP/JSON adapter boundary proven by the GAL API to Stratus
  `stratus.control` fleet-status path.

## Validation

- `npm run type-check`
- `npm test`
- `npm run build`
