# Agent Network v0.2.3

Adds the shared product issue gate evaluation contract for Business Ops Admin
product-status decisions.

## Changes

- Added `business-ops.product-issue-gate.evaluate` constants and JSON schemas.
- Added typed product issue gate evaluation, issue, product, classification, and
  enforcement shapes.
- Added `normalizeProductStatusIssueGateRecord(...)` so existing
  `product-status-issue-gate.v1` HTTP/JSON responses can be rendered through the
  same contract used by future Agent Network adapters.
