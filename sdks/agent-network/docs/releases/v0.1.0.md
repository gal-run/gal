# Agent Network v0.1.0

`v0.1.0` is the first internal Agent Network contract package release. It is a
private GitHub Release with an attached `npm pack` artifact. It is not a public
npm release.

## Included

- Enterprise Agent Card contracts.
- Service task lifecycle contracts.
- Health and status response contracts.
- Service SDK helpers for task creation, transitions, authorization checks,
  health/status responses, MCP tool descriptors, and A2A adapter metadata.
- Transport interoperability profile for A2A, gRPC, HTTP/JSON, GraphQL, MCP,
  and CLI adapters.
- Consumable package artifact checks for packed and git dependency installs.

## Compatibility

- Product APIs remain the source of truth for product behavior.
- GAL owns the first runtime/control-plane implementation.
- Product repos, including Stratus, own adapters over their own APIs.
- Runtime `/healthz`, `/readyz`, and `/status` endpoints are downstream
  implementation responsibilities that use the exported contracts.
