# v0.3.0

Adds GitHub dependency-map ingestion.

## Included

- GitHub dependency source schema for issues, PRs, workflow checks, releases, deployment gates, and external services
- conversion from GitHub dependency nodes into `GalPredictionRequest`
- blocker inference for review, CI, deployment, release, dependency, and external-service blockers
- CI profile inference from pending and failing checks
- cross-repo dependency validation and cycle detection
- tests proving dependency maps feed the existing forecast schema

- `npm install --package-lock-only`
- `npm run type-check`
- `npm test`
- `npm run build`
- `npm run smoke:consumer`
