# v0.2.0

Adds execution trace calibration inputs.

## Included

- execution trace schema for token burn, tool calls, CI queue/runtime, reruns, failures, and blockers
- trace calibration schema
- deterministic trace aggregation helpers
- `applyGalPredictionCalibration` for task estimate calibration
- validation for trace shape, duplicate IDs, negative usage, and invalid timing
- preserves the v0.1 execution forecast schema boundary

## Validation

- `npm install --package-lock-only`
- `npm run type-check`
- `npm test`
- `npm run build`
- `npm run smoke:consumer`
- `npm pack --pack-destination /tmp`
