# v0.6.0

Adds prediction-side burst readiness gates for first paid swarm startup.

## Included

- `evaluateGalPredictionBurstReadiness`
- `gal.prediction-burst-readiness.v1` schema version
- readiness checks for runnable work, blocked task ratio, confidence, cluster capacity, context fit, billable minutes, and spend caps
- first paid burst preflight documentation with a 100-point checklist
