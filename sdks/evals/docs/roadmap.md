# Roadmap

## Phase 1: Local Evaluation Contract

- Generic suite and report schemas.
- CLI runner.
- Email triage synthetic suite.
- Field-level deployment gates.

## Phase 2: More Adapters

- Recorded-prediction adapter (score outputs captured from any system).
- Additional deterministic, rule-based task-family adapters.

## Phase 3: Richer Scoring

- More field kinds (set membership, fuzzy/normalized match, numeric tolerance).
- Per-tag and per-segment metric breakdowns.

## Phase 4: Integration Helpers

- Stable JSON report contract for external dashboards and CI.
- Helpers for comparing report runs over time.
