# GAL Swarm v0.1.0

Initial private release of the GAL burst-compute swarm contract package.

## Contract Surface

- Swarm plan schema version `gal.swarm-plan.v1`.
- Worker lease schema version `gal.swarm-worker-lease.v1`.
- Swarm decision schema version `gal.swarm-decision.v1`.
- Provider-neutral compute profiles for Stratus, RunPod, Crusoe, GitHub Actions,
  Kubernetes, local pools, and future providers.
- Priority classes for release-critical, user-facing, scheduled, and speculative work.
- Orchestration mode vocabulary inspired by Swarms architecture shapes.
- Deterministic first policy for scale-up, hold, drain, and shutdown decisions.

## Boundary

`@gal/swarm` plans and governs temporary burst capacity. It does not provision
RunPod, Crusoe, Stratus, Kubernetes, or GitHub Actions workers directly.
Provider adapters should implement those actions separately.
