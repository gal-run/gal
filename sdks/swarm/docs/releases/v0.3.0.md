# v0.3.0

Adds multi-provider burst integration planning.

## Included

- provider kinds for RunPod, Crusoe, GCP, AWS, and Azure
- provider lifecycle, billing, metrics, SDK, adapter package, and auth-secret profiles
- provider candidate cost estimation with startup, runtime, shutdown, and minimum billable seconds
- deterministic provider ranking and selection
- forecast-driven scale-up can choose the best ranked provider instead of only the cost snapshot provider
- README provider integration matrix

## Validation

- `npm install --package-lock-only`
- `npm run type-check`
- `npm test`
- `npm run build`
- `npm run smoke:consumer`
- `npm pack --pack-destination /tmp`
