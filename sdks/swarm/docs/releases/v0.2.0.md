# v0.2.0

Adds prediction-driven swarm planning.

## Included

- structural forecast input contracts compatible with `@gal/prediction`
- `buildGalSwarmLoadFromForecast`
- `planGalSwarmDecisionFromForecast`
- forecast-driven scale-up, drain, and shutdown tests
- manual CI dispatch support for release proof

## Validation

- `npm install --package-lock-only`
- `npm run type-check`
- `npm test`
- `npm run build`
- `npm run smoke:consumer`
- `npm pack --pack-destination /tmp`
