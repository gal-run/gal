# v0.3.3

Fixes first-burst cost planning and adds final startup interlocks.

## Included

- separates logical forecast workers from paid compute units with `logicalWorkersPerComputeUnit`
- provider ranking can price `desiredComputeUnits` instead of multiplying cluster cost by logical worker lanes
- scale-up decisions expose optional `desiredComputeUnits`
- `evaluateGalSwarmBurstPreflight` blocks unsafe startup before Stratus GPU providers provision capacity
- tests for one 8x H200 cluster carrying 32 logical lanes
