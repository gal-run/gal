# Security Policy

## Reporting a vulnerability

Please report security vulnerabilities **privately** using GitHub's
[private vulnerability reporting](https://docs.github.com/en/code-security/security-advisories/guidance-on-reporting-and-writing-information-about-vulnerabilities/privately-reporting-a-security-vulnerability):
open the repository's **Security** tab and click **Report a vulnerability**.

Please do not open a public issue for security reports.

We aim to acknowledge reports within a few business days and will keep you
updated as we investigate and address the issue.

## Supported versions

This project is in active development; security fixes target the latest
released version on the default branch.
